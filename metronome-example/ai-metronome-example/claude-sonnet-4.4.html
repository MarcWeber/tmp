How to use:
  • Upper case = 100% loudness, lower case = 50% loudness
  • Z = highest pitch (10000 Hz), a = lowest pitch (100 Hz)
  • . = silence (rest)
  • First line defines beats per measure
  • Each line is one measure with arbitrary subdivisions
  • Characters in a line divide the measure equally
